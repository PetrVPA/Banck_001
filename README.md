# Учебный проект

## Описание:
В данный моменнт проект содержит 6 функций скомпанованных по две в модуле
Превый модуль masks.py должен маскировать введенные пользователем данные либо счета либо карты
Маска для карты: 1234 12** **** 1234 
Маска для счета: **1234

Второй модуль widget.py принимает данные от пользователя и формирует ответ с типом каты и замаскированным номером карты 
или маскированным номером счета
Маска карты: Visa Platinum 7000 79** **** 6361
Маска счета: Счет: Счет **4305
Для работы модуль использует функции модуля masks.py
Вторая функция модуля преобразует американский формат даты в нормальный.

Третий модуль processing реализует прием списка словарей с информацие по картам, их статусом и датой операции
функция проверяет стаутус карты. Параметр проверки карты не обязательный по умолчанию должен иметь значение 'EXECUTED'
Вторая функция модуля сортирует аналогичный список словарей только в порядке дат операций, по умолчанию ПО УБЫВАНИЮ не 
обязательный параметр.

Четвертый модуль generators.py в котором реализованы три функции. Функция filter_by_currency реализует фильтрацию 
транзакций по используемой валюте. Функция transaction_descriptions выполняет фильтрацию по типу операций по счетам и 
картам. Функция card_number_generator генерирует номера кар в установленном формате начиная со стартового номера до 
конечного указанного. Например с 1 по 2 генерирует 0000 0000 0000 0001, 0000 0000 0000 0002.
